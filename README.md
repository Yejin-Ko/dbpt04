# 서울시 따릉이 정보
 ##### 서울 열린데이터 광장 사이트에서 공공자전거 대여소 정보와 대여소별 이용 정보 및 대여 정보를 이용하여 정부 데이터를 공공에게 게시하고 도시의 삶을 좀 더 편리하고 가치있게 만들어 주는 시빅 해킹 프로젝트 입니다.
 
 ##### 기본적인 따릉이 대여소 정보를 출력하는 기능과, 따릉이의 일일권 대여 기록을 통해 자전거를 타기 좋은곳을 추천하는 기능이 있습니다.
